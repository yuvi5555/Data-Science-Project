# Data Science & Machine Learning Projects

This repository contains multiple Data Science and Machine Learning projects implemented using Python and Jupyter Notebook. These projects demonstrate practical applications of machine learning algorithms for classification, prediction, and recommendation systems.

Author: **Yuvraj Rajure**  
GitHub: https://github.com/yuvi5555

---

## Repository Description

This repository includes beginner to intermediate level machine learning projects such as Heart Disease Prediction, Spam Mail Detection, and a Movie Recommendation System. The projects showcase data preprocessing, feature engineering, model training, and evaluation using popular Python libraries.

---

## Projects Included

### 1. Heart Disease Prediction
This project predicts whether a person is likely to have heart disease based on medical attributes.

**Techniques Used**
- Data preprocessing
- Feature selection
- Machine learning classification

**Algorithms**
- Logistic Regression
- Scikit-learn models

**Objective**
To build a predictive model that helps identify heart disease risk using patient health data.

---

### 2. Spam Mail Prediction
This project classifies emails as **Spam or Non-Spam** using machine learning techniques.

**Dataset**
mail_data.csv

**Techniques Used**
- Text preprocessing
- TF-IDF vectorization
- Classification algorithms

**Algorithm**
- Naive Bayes / Logistic Regression

**Objective**
To automatically detect spam emails and improve email filtering systems.

---

### 3. Movie Recommendation System
This project recommends movies to users based on similarity between movies.

**Dataset**
movies_dataset.csv

**Techniques Used**
- Data preprocessing
- Feature extraction
- Cosine similarity

**Objective**
To recommend similar movies based on user preferences or movie features.

## How to Run the Projects

### 1. Clone the repository

```bash
git clone https://github.com/yuvi5555/Data-Science-Machine-Learning-Projects.git
```

### 2. Navigate to the project folder

```bash
cd Data-Science-Machine-Learning-Projects
```

### 3. Install required libraries

```bash
pip install numpy pandas scikit-learn matplotlib seaborn
```

### 4. Open Jupyter Notebook

```bash
jupyter notebook
```

Run the notebooks to explore each project.
